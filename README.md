# Портфолио: AI MAGIC WORDS

Одностраничный сайт-портфолио на чистом HTML/CSS/JS.

## Что внутри

- `index.html` — структура страницы и скрипты интерактивности.
- `styles.css` — стили, адаптив, анимации.
- Медиафайлы (`.png`, `.mp4`) — контент для hero, кейсов и галерей.

## Как запустить

1. Откройте папку проекта в Cursor/VS Code.
2. Запустите `index.html` через Live Server  
   или просто откройте файл в браузере.

## Технологии

- HTML5
- CSS3
- Vanilla JavaScript

## Примечания

- Проект не использует React, Tailwind и сторонние JS-библиотеки.
- Анимации и интерактив реализованы нативно.
